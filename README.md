# 自転車レンタル料金シミュレータ

走行距離と速度を入力して、自転車のレンタル料金をシミュレーション📊

## ℹ️使い方

### [リンク🔗](/rent_cycle_calc.html)をクリック

1. **数字を入力**

   * 走る距離（km）
   * 進む速さ（km/h）
   * はじめの30分でかかる料金（円）
   * 追加で延長するときの時間（分）
   * 延長1回でかかる料金（円）

2. **「料金を計算」をクリック**

   * 必要な時間と、2つのプランの金額を自動で計算
